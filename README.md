BURN 🔴
Aplicación web de entrenamiento (ES/EN) para fuerza, hipertrofia, calistenia y resistencia. Todo corre en un solo archivo HTML, sin instalación, y guarda los datos en tu propio dispositivo.
A bilingual (ES/EN) web training app for strength, hypertrophy, calisthenics and endurance. Runs from a single HTML file, no install, and stores data on your own device.
✨ Funciones · Features

Rutinas por día editables: agregar, renombrar, reordenar y borrar días.
Biblioteca de +200 ejercicios por equipamiento: máquina, polea, barra, mancuernas, bandas, peso corporal y calistenia.
Plantillas prearmadas por tipo (fuerza, hipertrofia, calistenia, resistencia) y 5 niveles de dificultad.
Series, reps, peso y descanso configurables, con registro por serie y descansos individuales.
Temporizador de descanso y cronómetro de sesión.
Modo entrenar a pantalla completa.
Progreso con historial y gráfico por ejercicio, más calculadora de 1RM.
Deshacer / Rehacer y propagar configuración a todos los ejercicios.
Frases motivacionales (incluye citas en latín) y dedicatoria editable.
Importar / Exportar: rutinas en .json (para compartir entre dispositivos), tablas con formato a Excel .xlsx, e importación desde Excel.
Fondo personalizable y tema rojo y blanco.

🚀 Uso · Usage
Abrí la app desde la URL publicada (GitHub Pages) o abriendo index.html en el navegador. Para mover una rutina de un dispositivo a otro, usá 📤 Compartir para bajar el .json y 📥 Importar para cargarlo.
Open the app from the published URL (GitHub Pages) or by opening index.html in your browser. To move a routine between devices, use 📤 Share to download the .json and 📥 Import to load it.
📊 Importar desde Excel · Excel import

Fila 1 (encabezados): los días (ej: Lunes, Miérc...).
Cada columna debajo de un día: un ejercicio por fila.
Celda con formato opcional Nombre  4x8 @30 r90 → 4 series, 8 reps, 30 kg, 90 s de descanso.

🛠️ Tecnología · Tech
HTML, CSS y JavaScript puro. Sin frameworks. El export/import de Excel usa SheetJS cargado por CDN (requiere conexión solo para esa función).
